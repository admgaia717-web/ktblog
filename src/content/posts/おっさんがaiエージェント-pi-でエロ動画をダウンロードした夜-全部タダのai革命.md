# おっさんがAIエージェント「pi」でエロ動画をダウンロードした夜——全部タダのAI革命

> 出典: note.com / 2026-04-30

## 55歳、AIエージェントにエロ動画をダウンロードさせた夜

告白する。

私は昨夜、AIコーディングエージェント「pi」に命じて、某アダルト動画サイトの無修正動画をダウンロードし、外付けSSDに保存させた。すべて黒い画面——ターミナル——の中だけで完結した操作だ。

そんな話をなぜここに書くかというと、これがもう「SF」ではなく「現実」であり、しかも**全部無料でできる**からだ。

## pi（パイ）——ターミナルに住むAI部下

piは npm install 一発でMac/Linuxに入るターミナル型AIエージェントだ。ChatGPTのようなチャット画面ではない。黒背景に白文字、いにしえのUNIXの世界である。

仕組みはシンプルだ。ユーザーが日本語で指示を出す → piがLLM（大規模言語モデル）に問い合わせる → LLMが「ファイルを読め」「コマンドを実行しろ」「コードを書け」と判断 → piがそれを実行して結果を返す。このループを自律的に繰り返す。

piがデフォルトで持つ能力は read（ファイル読み取り）、write（ファイル作成）、edit（ファイル編集）、bash（シェル実行）の4つ。これに加えて、拡張機能（Extensions）とスキル（Skills）で機能を無限に追加できる。

たとえばTelegram連携、X投稿、Appleメモ操作、YouTube文字起こし、ComfyUI画像生成、さらにはブラウザ自動操作。全部プラグインとして後付けできる。要するに「なんでも屋のAI部下」だ。

## 全部タダ。OpenRouterの無料モデルという革命

ここが一番重要なポイントだ。piの頭脳——LLM——はOpenRouterというAPI中継サービスを通じて切り替えられる。

OpenRouterが提供するモデルの多くに**無料枠**がある。2026年4月現在、以下のモデルが無料で使える：

• **DeepSeek V4 Flash** — 1Mトークンのコンテキストウィンドウ、高速推論
• **Qwen 3.6** — 262Kコンテキスト、中国発の高性能モデル
• **Ling 2.6** — 1Tコンテキスト（！）、超長文処理
• **Tencent HY3** — テンセントの最新モデル
• **Nvidia Nemotron Nano** — 推論特化の30Bモデル

ChatGPT Plusのような月額3000円は不要。OpenRouterでAPIキーを取得し（無料）、piに設定するだけ。1円も払わずに、プロのエンジニア並みの作業をAIが代行してくれる。これは革命だ。

さらにpiはローカルのOllamaモデルも使える。私はM4 Mac miniにQwen 3.6 27B（無修正版）とGemma4 7.5B（無修正版）を常駐させている。インターネット不要、検閲なし、完全無料。地方在住の55歳が個人でここまでできる時代になった。

## 実況：AIと二人三脚で動画を落とすまで

実際の流れを技術的に詳述する。

**Step 1 — リサーチ**
piに「このMissAVのURLから動画をダウンロードして」と指示。piはまずAgentReach（Web検索ツール）で「missav.ai download m3u8」を検索。GitHubから専用ツール「missav-dlw」「miyuki」「missAV_api」の3つを発見し、それぞれのPros/Consを比較表にまとめてきた。

**Step 2 — 試行錯誤**
最初にmissav-dlw（Pythonの非同期HLSダウンローダー）を試す。が、surrit.comのCDNから403 Forbiddenが返る。ここでpiは諦めず、curlにRefererヘッダーを追加して再試行。成功。m3u8プレイリストの取得に成功。

**Step 3 — ffmpegの罠**
取得したm3u8をffmpegに直接渡すも、「video0.jpeg is not in allowed segment extensions」で拒否。MissAVのセグメントファイルは拡張子が.jpegだが実体はMPEG-TSという特殊仕様だった。

**Step 4 — 自作スクリプト**
piが即席で書いたPythonスクリプト：m3u8をパース→711個の.jpegセグメントを並列ダウンロード→ffmpeg concatデマルチプレクサでMP4に結合。15分で488MBのファイルが外付けSSDに保存された。

**Step 5 — 後始末**
ファイル名を「HEYZO-2464_1年ぶりに爆乳美少女と再会_20260430.mp4」にリネーム。タイムスタンプ付きで人間可読。マグネットリンクでtorrent取得を試みたがピア不在で失敗したのも含めて、piはすべての試行錯誤を記録していた。

## 実用性の証明として

この一件が示すのは「エロ動画をダウンロードできる」という表層的な話ではない。

piのようなAIエージェントは、技術的な障壁を取り払う。m3u8ストリームの構造を知らなくても、HLSの仕組みを理解していなくても、Cloudflareのブロックを回避する方法がわからなくても、「これやって」の一言で突破できる。

これは単なる動画ダウンロードの話ではなく、「あらゆるデジタル作業がAIで自動化できる」という証明である。

私は55歳。奈良在住。プログラマーではない。AIと暗号通貨と仏教哲学を趣味とする、ただのおっさんだ。そんな私が、無料のAIだけでここまでできる。

もちろん「AIにこんなことさせていいのか」という倫理的問いはある。でも同時に思う——新しい道具が出てきたとき、人はまず遊び、そして可能性を見出し、やがて社会が追いつく。インターネットもスマホも、最初はみんなエロから入った。歴史は繰り返す。

## piを始める3ステップ

**1. インストール（30秒）**

npm install -g @mariozechner/pi-coding-agent

**2. OpenRouterでAPIキー取得（無料・1分）**

openrouter.ai にGitHubアカウントでログイン → 「Create Key」をクリック → キーをコピー。

**3. piにキーを設定して起動**

export OPENROUTER_API_KEY=sk-or-v1-xxxxx
pi

あとは日本語で話しかけるだけ。Slackの有能な後輩に頼む感覚でいい。

55歳のおっさんが夜な夜なAIと二人三脚でエロ動画をダウンロードしている——そういう時代になった。
