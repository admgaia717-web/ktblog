# 音声でベラベラ喋っただけ——AIが勝手にnote記事を書いて公開した。全部タダだ

> 出典: note.com / 2026-04-30

## 音声でベラベラ喋っただけなのに、AIが勝手にnote記事にして公開した

これ、今この瞬間に起きていることである。俺は今、キーボードに一切触っていない。Macの前に座って、音声入力でベラベラと喋っているだけだ。「おい、pi。この雑談をいい感じのnote記事にしといて。アイキャッチも作って。ついでに公開までやっといて」——それだけ言った。

するとpi（AIコーディングエージェント）が勝手に動き出す。① 俺の音声を文字起こし ② ダラダラした話を構造化して読みやすい文章に再構成 ③ Stable Diffusionでアイキャッチ画像を自動生成 ④ note.comのAPIを叩いて下書き→アイキャッチ設定→公開。この4工程、全部AIがやった。人間は「これ、記事にしといて」と言っただけ。執筆時間ゼロ秒。編集時間ゼロ秒。画像選びゼロ秒。これが2026年4月の現実だ。

## 1月は月10万円払ってもできなかった

思い返せば今年の1月、俺は狂ったようにAIに金を払っていた。Claude Codeに月3万円。OpenAIのChatGPT Proに月3万円。Alibaba Codingに数千円。その他いろいろAPI使用料で数千円。合計で月に7〜8万円。それでも「文章的には微妙」「コードがバグる」「途中で止まる」の連続。ストレスしかなかった。

ところがどうだ。2026年4月。今、俺は**1円も払っていない**。OpenRouterの無料枠モデル——DeepSeek V4 Flash、Qwen 3.6、Ling 2.6、Tencent HY3——をローテーションで使っている。ローカルにはM4 Mac miniでQwen 27B（無修正版）が常駐している。全部タダ。なのに1月の有料版より遥かに高性能。どうなってるんだこれは。

![燃える札束](https://assets.st-note.com/production/uploads/images/271933601/rectangle_large_type_2_d8cf16c9f00d1e56ac24092c34fcfc19.png)

## 具体的にどうやってるのか

**1. pi（エージェント）** npm install -g @mariozechner/pi-coding-agent で入る。ターミナルの中で動くAI。ファイル読み取り、コマンド実行、コード生成、Web検索、ブラウザ操作を自律的にやってくれる。

**2. OpenRouter（LLM中継）** APIキーひとつで30以上のモデルにアクセスできる。しかも無料枠が充実。DeepSeek、Qwen、Ling、Tencent...毎月新しいモデルが追加される。

**3. ComfyUI + SDXL Lightning（画像生成）** ローカルMacで画像生成。4ステップ・22秒でアイキャッチができる。プロンプトもAIが書くから人間は「いい感じの絵つけといて」で終わり。

**4. note.com非公式API（投稿）** curlでnoteにログイン→記事作成→画像アップロード→公開。ブラウザ不要。この部分もAIがコードを書いて自動化した。

つなげるとこうなる：人間「これ記事にしといて」→ pi「了解。まず音声を文字起こしして、構造化して、アイキャッチ生成して、noteに投稿します」→ 数分後「公開完了しました。URLはこれです」終わり。人間の作業は声を出しただけ。

## やばい。本当にやばい

正直、興奮している。だって考えてみろ。個人が、地方在住の55歳のおっさんが、月額0円でここまでできるんだぜ。朝起きて「今日のAIニュースまとめといて」と言えばpiがWeb検索→要約→note記事化→X投稿までやる。YouTubeのURLを投げれば文字起こしからブログ化までやる。これ、会社の新人に仕事を振るより速い。しかも文句を言わない。24時間働く。給料はいらない。

## ロボットまで来る

今やAmazonでRaspberry Piとサーボモーターと3Dプリンタが買える。ROSはオープンソース。AIがコードを書く。piが実装する。つまり「おい、pi。Amazonでロボットアームのパーツ頼んで、組み立て手順書いて、制御コードも書いといて」——これが現実にできる。

個人が、自宅で、無料のAIと格安の部品だけで、物理的なロボットを作れる時代。1月は月10万円払ってもできなかったことが、4月には0円でできる。5月には何ができる？6月には？この進化の速度、マジで頭がおかしい。

## お前ら、これで興奮しないのか

いや、するだろ。AIがコードを書き、AIが画像を生成し、AIが記事を公開し、AIがロボットを制御する。その司令塔になるのがpiのようなエージェントだ。しかも全部タダ。

今、この文章を読んでいるあなたも、今日からできる。npm install -g @mariozechner/pi-coding-agent。OpenRouterでAPIキーを取る（1分・無料）。piを起動。あとは日本語で話しかけるだけ。さあ、一緒にこのクレイジーな時代を楽しもうぜ。
