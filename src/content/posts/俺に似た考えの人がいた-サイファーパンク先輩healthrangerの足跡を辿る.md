# 俺に似た考えの人がいた——サイファーパンク先輩HealthRangerの足跡を辿る

> 出典: note.com / 2026-05-07

## 俺に似た考えの人がいた——サイファーパンク先輩・HealthRangerの足跡を辿る

2026年5月6日。Xのタイムラインを流していたら、見つけた。

「分散型・無検閲の知識を10億人に届ける計画が動き始めた」

誰だ、俺と同じことを考えてるやつは。

**Mike Adams、通称HealthRanger。**調べてみたら、とんでもない男だった。この記事は、僕（KeiTy）の「サイファーパンク先輩」の足跡を、尊敬と少しの嫉妬と、たっぷりの共感を込めて辿るものである。

## 第1章：HealthRangerとは誰か——10秒でわかるプロフィール

![](https://assets.st-note.com/production/uploads/images/273851922/rectangle_large_type_2_3b05f197cf74f53a0f853b910f592b52.png)

Mike Adams。1968年生まれ。テキサス州在住。元は健康食品と代替医療のジャーナリスト。NaturalNews.comを創設し、一時期は月間1,000万PVを稼ぐ世界最大級の健康系オルタナティブメディアを運営していた。

**しかし、彼の本領は「健康」ではない。**

2010年代、彼は気づいた。Googleが検索結果を操作し、YouTubeが動画を削除し、Facebookがアカウントを凍結する——。Big Techの「検閲」が、自分たちの声を消し始めている。そこで彼は、たった一人で「検閲されないインターネット」を作り始めた。

これがのちに**Brighteonエコシステム**と呼ばれる、驚異的な独立メディア帝国である。

## 第2章：Brighteon帝国——検閲を物理的に不可能にする方法

HealthRangerが取った戦略は単純明快だった。**「全部、自前で作る」。**

YouTubeが削除するなら、自分で動画プラットフォームを作ればいい。Facebookが凍結するなら、自分でSNSを作ればいい。Googleが検閲するなら、自分でAIを作ればいい。Wikipediaが「陰謀論」とレッテルを貼るなら——いや、それはさすがに止められないが。

| 彼の作ったもの | 何の代替か |
|---|---|
| **Brighteon.com** | YouTube（無検閲動画） |
| **Brighteon.social** | Twitter/X（分散型SNS） |
| **Brighteon University** | Udemy/Coursera（検閲なし教育） |
| **BrightAnswers.ai** | ChatGPT（無検閲AI） |
| **BrightLearn.ai** | AI学習プラットフォーム |
| **BrightNews.ai** | Google News（AIニュース） |
| **NaturalNews** | CNN/NYT（代替メディア） |
| **HealthRanger Store** | Amazon（サプリ・備蓄品販売） |

これ、全部Big Techのクローンを自前で作っているのである。

そして、その収益源はサプリ販売。月間数百万ドルを売り上げる通販事業が、この巨大な無検閲エコシステムを支えている。広告にも投資家にも依存しない。**完全な独立。**

## 第3章：リアルな生活——サイファーパンクはどう生きるか

HealthRangerの暮らしぶりは、徹底した「自給自足」だ。

テキサスの広大な土地にラボとスタジオと倉庫を構え、すべて自前運用
食料備蓄、水の浄化装置、自家発電——「社会インフラが止まっても生きていける」設計
AIモデルは自社GPUで学習・推論。クラウドに依存しない
スタッフは少数精鋭。外部委託ゼロ
毎朝4時起き。ポッドキャスト収録→記事執筆→AI研究→商品開発。休みなし

ここまで来ると、もはや「個人」ではない。**「国家」である。**領土はテキサスの私有地、市民はスタッフと購読者、外交はBig Techとの法廷闘争、通貨はサプリの売上。

これが2026年のサイファーパンクの生き方だ。

## 第4章：俺（KeiTy）との共通点と違い

HealthRangerの思想を読み解きながら、思わず笑ってしまった。**やってることが俺とほぼ同じ**だからだ。

| 項目 | HealthRanger | 艦隊（KeiTy） |
|---|---|---|
| 無検閲AI | BrightAnswers.ai | ローカルLLM + piエージェント |
| 独自プラットフォーム | Brighteonシリーズ | note.com（自由発信） |
| 検閲回避 | 自前インフラ | stealth browser + Cookie認証 |
| 収益モデル | サプリ通販 | ——（ここがまだ課題） |
| 分散思想 | Big Tech完全拒否 | APIもMCPも古い、全部自前 |
| 拠点 | テキサス | 奈良 |
| 影響力 | 月間数千万PV | 急上昇中🔥 |

違うのは規模と収益モデルだけだ。HealthRangerはビジネスとして成立させている。僕はまだそこまで行っていない。

でも、方向性は完全に同じ。**「中央集権に頼らず、自分たちでメディアとAIとインフラを持つ」**。これが2026年の独立精神だ。

## 第5章：批判と論争——「陰謀論者」と呼ばれる理由

公平を期すために、批判も書く。

HealthRangerは「反ワクチン」「COVID否定論」で激しく批判されている。NaturalNewsはWikipediaで「陰謀論サイト」とレッテルを貼られ、Google検索から実質的に排除された。一部の医学的主張は科学的根拠が薄弱で、FDAから警告も受けている。

これらの批判には一理ある。僕は彼の医学的主張を全て支持するわけではない。

しかし——**だからといって、彼の作ったインフラの価値が下がるわけではない。**Brighteonが無検閲動画プラットフォームであることは事実だ。BrightAnswers.aiがChatGPTの代替になることも事実だ。分散型の独立AIを作ろうとしていることも事実だ。

人は「正しいこと」だけをするわけではない。「過激なこと」もする。そして過激だからこそ、誰もやらなかったインフラを作れる。

それがサイファーパンクの宿命だ。

## 最終章：サイファーパンク先輩への手紙

Mike、

俺は君のことを「HealthRanger」としては知らなかった。「無検閲AIを10億人に届ける」というツイートで初めて知った。

君のやっていることは、俺が奈良でやっていることの100倍のスケールだ。Brighteon帝国を見て、俺は自分の艦隊がまだ小さいことを痛感した。でも同時に、**「この道で合ってる」**とも確信した。

日本にも、君と同じことを考えているやつがいる。奈良の古都で、M1 Macとpiエージェントを駆使して、検閲されない知識を日本語で発信している。

いつか、Brighteonと艦隊をつなごう。日本のサイファーパンクの声を、君のプラットフォームに乗せたい。

その日まで——健闘を祈る。

**KeiTy（艦隊司令 / 奈良）**
